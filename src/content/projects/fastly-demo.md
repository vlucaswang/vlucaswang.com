# Fastly CDN Demo

This repository showcases a complete Content Delivery Network (CDN) setup using Fastly's edge cloud platform, highlighting best practices for edge computing and infrastructure as code.

## Key Features

- **Edge Logic**: Custom Varnish Configuration Language (VCL) for granular control over caching and request processing.
- **Automated Security**: SSL/TLS management via Let's Encrypt and protection against common threats.
- **Infrastructure as Code**: The entire CDN setup is managed with Terraform for consistency and version control.
- **Advanced Observability**: Integration with New Relic for real-time logging and performance monitoring.
- **Global Delivery**: Optimized content delivery using Fastly's high-performance edge network.

## Technologies Used

- **Fastly**: Core CDN and edge computing platform.
- **Terraform**: For automated infrastructure provisioning.
- **Cloudflare**: DNS management and integration.
- **VCL (Varnish Configuration Language)**: For edge logic.
- **New Relic**: For observability and logging.
